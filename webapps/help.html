Search Cloud Help


什么是search cloud

统计管理和配置操作solr，你不用操心solr的环境，不用操心每次新建solr要做很多重复的操作，也不用担心每次操作还要跑到服务器上去弄，现在search cloud一起都帮你搞定

什么是service

一个service代表一个搜索服务，比如二手房房源列表的搜索就可以是一个搜索服务，对应着一个service

什么是instance

instance是service下实例，用来处理service接受到的搜索请求，

什么是Host

Host对应一台真实的服务器，instance的实例都跑在上面

service、instance、host的关系是怎样的

处理各种搜索请求的都是instance，service相当于instance的虚拟集合，instance都跑在host上，一个service的各个instance可以分步在不同的host上

如何新建service

在service list有新建service的入口，然后到新建service的三步引导，顺着引导就可以轻松创建service了

如何新建instance

在instance list页面和service detail页的instance list页都有新建instance的入口，按引导完成创建即可

如何管理service

service detail首页就有对service各种操作的按钮

如何管理instance

instance list可直接对instance进行管理，你也可以到service单页对instance进行管理
